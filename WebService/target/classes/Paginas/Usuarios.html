Usuários

Caminhos:

/usuarios/listar - Neste é possível listar todos os usuários já cadastrados no banco de dados

/usuarios/cadastrar/{usuario}/{senha} - Neste é possível cadastrar um usuário enviando como parâmetro o nome do usuário e a senha

/usuarios/logar/{usuario}/{senha} - Neste é possível logar um usuário enviando como parâmetros o nome do usuário e a senha

Caso a autenticação de login ou de cadastro seja possível, um resultado será emitido

Todas as informações retornadas estão em formado JSON (JavaScript Object Notation)
